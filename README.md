# 東証 最強BOTシミュレーター (auto-trade)

機関投資家やプロのアルゴリズムトレーダーが実際に用いる「テクニカルとファンダメンタルズの融合」を実装した、完全自律型の日本株シミュレーショントレードBOTです。
東証の全銘柄（約3800銘柄）から資金流入の初動を検知し、AIがテーマ性やニュースを加味して自動で仮想売買・資金管理を行います。

## 🔥 特徴・搭載ロジック

### 1. 高精度テクニカルスクリーニング

東証全銘柄の15分足データを高速スキャンし、人間が見逃す「急騰の初動」を以下の条件で機械的に足切りします。

* **資金流入検知**: 過去の平均出来高に対して3倍以上の出来高急増
* **ボラティリティ・ブレイク**: ボリンジャーバンドのスクイーズ（収縮）からのエクスパンション（急拡大）
* **トレンドブレイク**: 直近高値の更新
* **VWAPフィルター**: 現在値がVWAP（出来高加重平均価格）を上回っているか（騙し下げの回避）
* **上ヒゲ回避**: 高値掴みを防ぐため、ATR（ボラティリティ）を基準に長すぎる上ヒゲを排除

### 2. 生成AIによるプロレベルのスコアリング

スクリーニングを通過した「資金流入率上位50銘柄」に対し、Googleの最新AI（Gemini 2.5 Flash）がプロの機関投資家として振る舞い、1〜100点でスコアリングします。

* テクニカル指標（RSI, MACD等）と、取得した最新ニュース（Google News RSS）を統合分析。
* AI、半導体、防衛などの「テーマ性」、決算サプライズ、SNSの話題性を加味して騙しを回避します。
* API制限（429エラー）を自動検知し、待機・再試行する堅牢なフェイルセーフ機能を搭載。

### 3. リスク管理と利益の極大化

* **地合いフィルター**: 日経平均（^N225）の15分足チャートを確認し、短期下落トレンド時は新規エントリーを見送ります。
* **トレーリングストップ**: 含み益が+2%を超えたら利確準備に入り、その後最高値から-2%下落した瞬間に「トレール利確」を実行。利益を最大限に伸ばします。
* **自動損切り**: -3%に達した時点で機械的に損切りを実行します。
* **複利運用**: 口座残高を管理し、利確・損切りの結果を反映。残高の範囲内で買えるだけ買い付けます。

## 📁 ファイル構成と見方

本プログラムを実行すると、自動的に複数のファイルが生成・更新されます。

| ファイル名 | 役割・見方 |
| --- | --- |
| `auto_trade.py` | BOTのメインプログラム。このスクリプトを定期実行します。 |
| `data_j.csv` | **【入力】** 東証の上場銘柄リスト。コードや銘柄名が記載されています。 |
| `account.json` | **【自動生成】** 現在の仮想口座の残高を記録するファイルです。（初期資金：100万円） |
| `virtual_portfolio.csv` | **【自動生成】** 現在「保有している（買った）」銘柄のリスト。買値や株数、記録上の最高値が記載されます。 |
| `trade_history.csv` | **【自動生成】** 売却（利確・損切り）が完了した取引の履歴ファイル。勝率や累計損益の分析に使用します。 |

## 🚀 使い方・セットアップ

### 1. 必要なライブラリのインストール

Python環境が構築されているPCで、以下のコマンドを実行し必要なパッケージをインストールします。

```bash
pip install yfinance pandas numpy google-genai feedparser

```

### 2. APIキーの設定

`auto_trade.py` のコード内にある以下の部分に、取得したGeminiのAPIキーを入力してください。

```python
GEMINI_API_KEY = "ここにAPIキーを入力"

```

### 3. プログラムの実行

コマンドプロンプトやVSCode等からスクリプトを実行します。

```bash
python auto_trade.py

```

### 4. 完全自動化（タスクスケジューラの設定）

Windowsの「タスクスケジューラ」を利用し、本スクリプトを **1時間ごと（例：毎時0分）** に自動実行するよう設定することで、完全放置のシミュレーション環境が完成します。
※AI審査の上限を50銘柄に設定しているため、1時間ごとの実行であればGemini APIの無料枠（1分15回、1日1500回）の範囲内で安全に稼働します。

## ⚙️ パラメーター設定（カスタマイズ）

運用結果を見てロジックを調整したい場合は、`auto_trade.py` 上部の以下の変数を変更してください。

```python
INITIAL_CASH = 1000000       # 初期の仮想資金（円）
MAX_POSITIONS = 3            # 同時に保有できる最大銘柄数
INVEST_PER_TRADE = 300000    # 1銘柄あたりの投資上限目安（円）
STOP_LOSS_PCT = -0.03        # 損切りライン（-3%）

TRAIL_ACTIVATION_PCT = 0.02  # トレーリングストップ発動ライン（含み益 +2%）
TRAILING_STOP_PCT = 0.02     # 最高値からの下落許容幅（-2%）

MAX_AI_CANDIDATES = 50       # 1回の実行でAIに審査させる最大銘柄数

```
