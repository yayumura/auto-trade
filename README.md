# 🏦 ヘッジファンド仕様・究極のAI自律トレードエンジン (Zero-Base Audit Passed)

機関投資家レベルの**マルチファクター＆レジームスイッチング戦略**を搭載し、全14フェーズの「ゼロベース徹底監査」を完遂した、日本株完全自動売買システムです。

単なる売買スクリプトではなく、**異常検知・自己修復・取引所同期・リソース監視**までを備えた、「24時間365日の安定稼働」を前提とした運用プラットフォームへと進化しました。

---

## 💎 プロフェッショナル仕様への進化 (Zero-Base Audit)

本システムは、実弾運用におけるあらゆるリスクを排除するため、全14段階の監査防衛層を構築しました。

1.  **自律ループ型エンジン (`auto_trade.py`)**: 08:50に一度起動すれば、15:30の大引けまで自律的に監視・判断・売却を繰り返し、大引け後に自動終了します。
2.  **取引所サーバー時刻同期**: PCの時計のズレに左右されず、証券会社のサーバー時刻に秒単位で同期。
3.  **未約定注文保護 (In-flight Guard)**: 起動時に「執行中の注文」をAPIで確認し、二重発注を物理的に防止。
4.  **価格入力の階層化 (Price Source Hierarchy)**: スキャンには広範な yfinance、損切り判定には API の「リアルタイム秒速価格」を優先使用。
5.  **株式分割自動同期**: API側の実データとローカル記録を自動ですり合わせ、分割時の資産計算ミスを排除。
6.  **自動清掃 (Housekeeping)**: 肥大化するキャッシュファイル（yfinanceデータ等）を7日分に自動クリーニング。

## 🔥 搭載ロジック

### 1. レジーム（相場環境）判定
日経平均のデータから現在の相場環境を自動判定。
- **BULL（強気）**: モメンタム戦略（順張り）で利益を最大化。
- **RANGE（もみ合い）**: ミーンリバージョン戦略（逆張り）で安値を拾う。
- **BEAR（パニック）**: **新規買付を物理的にロック**し、保有ポジションの出口戦略のみ実行。

### 2. 数学的スクリーニング & AI判定
- **数学フィルター**: 4000銘柄を瞬時にスキャンし、出来高・MACD・RVI・VWAP等から上位候補を抽出。
- **AI防護壁**: 厳選された候補のみ、Gemini 1.5 Flash（またはGroq）が悪材料ニュースを最終審議。

## 🚀 セットアップと実行

### 1. 環境構築
```bash
# 依存関係を一括インストール
pip install -r requirements.txt
```

### 2. 環境変数の設定 (.env)
```env
GEMINI_API_KEY=your_key
GROQ_API_KEY=your_key
DISCORD_WEBHOOK_URL=your_webhook
KABUCOM_API_PASSWORD=your_password
TRADE_MODE=SIMULATION # または KABUCOM_TEST / KABUCOM_LIVE
```

### 3. 運用方法（推奨）
Windowsタスクスケジューラで、**平日の朝 08:50** に1回だけ `auto_trade.py` を起動するように設定してください。

- **起動**: `python auto_trade.py`
- **終了**: 15:30に自動終了。手動停止は `Ctrl+C`。

## ⚙️ 主要パラメータ

（`core/config.py` 内で変更可能です）

| パラメータ | デフォルト値 | 説明 |
|:---|:---|:---|
| `MAX_POSITIONS` | 4 | 最大同時保有銘柄数 |
| `MAX_ALLOCATION_PCT` | 30% | 1銘柄あたりの最大投資額（対総資産） |
| `MAX_RISK_PER_TRADE` | 2% | 1トレードあたりの許容リスク（ATRベース） |
| `ATR_STOP_LOSS` | 2.0 | 損切り係数（順張り時） |
| `ATR_TRAIL` | 1.5 | トレーリングストップ係数 |

## 🛡️ 究極の安全機構
- ✅ **起動前自己診断 (Pre-flight Check)**: 接続・設定・権限を100%確認してから稼働。
- ✅ **寄り付き30分回避**: 9:00〜9:30の混沌とした値動きへのエントリーを禁止。
- ✅ **大引け前エントリー禁止**: 14:30以降の新規買付を抑制し、引け際のリスクを回避。
- ✅ **価格の Data Age Guard**: 古いデータ（1時間以上前）による誤判定を遮断。
- ✅ **エラークールダウン**: API制限や一時的なエラー発生時、10秒間の静止で連鎖崩壊を防止。
- ✅ **優雅なシャットダウン**: 停止信号を検知し、安全に終了。
