# 東証 最強BOTシミュレーター (初動ブレイクアウト完成版)

機関投資家やプロのクオンツファンドが実際に用いる「テクニカルとファンダメンタルズの融合」を実装した、完全自律型の日本株シミュレーショントレードBOTです。
東証の全銘柄（約4000銘柄）から資金流入の初動（15分足）を検知し、ハイブリッドAI（Gemini & Groq）がテーマ性やニュースを加味して自動で精査・仮想売買・資金管理を行います。

今回の「完成版」では、順張りブレイクアウト最大の弱点である**「高値掴み（イナゴタワー）」**と**「微細なノイズによる同値撤退」**を完全に克服する鉄壁のフィルターを搭載しています。

## 🔥 特徴・搭載ロジック

### 1. 高精度テクニカルスクリーニング (15分足)
東証全銘柄の15分足データを高速スキャンし、人間が見逃す「急騰の初動」を以下の条件で機械的に足切りします。
* **流動性・低位株フィルター**: 株価100円未満、または15分平均売買代金が1000万円未満の過疎株・クズ株を排除し、スリッページを防ぎます。
* **資金流入検知**: 過去の平均出来高に対して3倍以上の出来高急増（大口の介入）。
* **ボラティリティ・ブレイク**: ボリンジャーバンドのスクイーズ（収縮）からのエクスパンション（急拡大）または直近高値更新。
* **【鉄壁】高値掴み絶対防御**: 現在値がVWAP（出来高加重平均価格）から**+5%未満**の乖離に収まっており、かつ**RSIが75未満**の「まだ上がりきっていない安全な初動」のみを抽出します。
* **上ヒゲ回避**: 高値掴みを防ぐため、ATR（ボラティリティ）を基準に長すぎる上ヒゲを排除。

### 2. ハイブリッドAIスコアリング（Gemini & Groq）
スクリーニングを通過した「資金流入率上位50銘柄」に対し、AIがプロの機関投資家として振る舞い、1〜100点で厳格にスコアリングします。
* **テーマ性と材料の評価**: 取得した最新ニュース（Google News）から、「資金流入の信憑性」「テーマの持続性」を評価し、ニュースのない仕手株のダマシを減点します。
* **APIフォールバックシステム**: メインのAI（Gemini 2.5 Flash）がレートリミット（利用制限）に達した場合、自動的に超高速・無料のAI（Groq Llama 3.3）に通信を切り替えて審査を続行する、無停止のハイブリッドシステムを搭載。

### 3. プロ仕様のリスク管理と資金管理
* **最適化された動的トレーリングストップ（ATR）**: 固定のパーセンテージではなく、銘柄ごとのボラティリティ（ATR）を基準に利確・損切りラインを自動計算。15分足のノイズで狩られないよう、**損切りをATRの2.5倍、利確準備を3.0倍**と余裕を持たせ、大きなトレンドの波に乗ります。
* **リスク分散投資**: 資金100万円に対し、最大保有銘柄数を「4」、1銘柄の投資上限を「25万円」に設定し、1回の負けによる致命傷を防ぎます（予算内で1株単位で限界まで買い付けます）。
* **税金計算システム**: 利益が出た決済に対しては、実際の特定口座（源泉徴収あり）と同様に**20.315%の税金**を自動的に差し引いて口座残高に戻すため、極めてリアルな資産推移が計測できます。
* **地合いフィルター**: 日経平均（^N225）の15分足チャートを確認し、短期下落トレンド時は新規エントリーを一時見送ります。

### 4. 充実の通知・ログ監視機能
* **Discordリアルタイム通知**: 新規買付や利確・損切りが発生した瞬間に、スマホのDiscordへ詳細なメッセージを自動送信します。
* **実行結果サマリー＆資産推移記録**: プログラム終了時に現在の口座残高や保有株式の含み損益をサマリー表示し、同時に `execution_log.csv` へ資産推移を自動記録します。
* **詳細ログの自動保存**: ターミナルに出力されるすべての情報（AIの判定理由など）を、日別のログファイル（`logs/console_YYYY-MM-DD.log`）として自動保存し、後からAIの癖や動作を詳細に検証できます。

### 5. 完全自律稼働（営業時間フィルター）
* プログラム自身が現在時刻を取得し、**平日の東証取引時間（9:00〜11:30、12:30〜15:30）のみ稼働**します。夜間や土日に無駄にAPI枠を消費することはありません。

## 📁 ファイル構成と見方

| ファイル名 | 役割・見方 |
| :--- | :--- |
| `auto_trade.py` | BOTのメインプログラム。このスクリプトを定期実行します。 |
| `.env` | **【ユーザー作成】** APIキーやDiscordのWebhook URLを安全に保存するためのファイル。 |
| `data_j.csv` | 東証の上場銘柄リスト。コードや銘柄名が記載されています。 |
| `account.json` | **【自動生成】** 現在の仮想口座の現金残高を記録するファイルです（初期資金：100万円）。 |
| `virtual_portfolio.csv` | **【自動生成】** 現在保有している銘柄のリスト。買値や株数、最高値などが記載されます。 |
| `trade_history.csv` | **【自動生成】** 売却が完了した取引履歴。勝率や税引き後の純利益などが記載されます。 |
| `execution_log.csv` | **【自動生成】** 実行ごとの現金・株式評価額・合計資産額の推移を記録します。資産カーブの分析に使用します。 |
| `logs/` フォルダ | **【自動生成】** 日別（`console_YYYY-MM-DD.log`）でコンソールの出力内容がすべて保存されます。 |
