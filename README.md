# 東証 最強BOTシミュレーター (auto-trade)

機関投資家やプロのクオンツファンドが実際に用いる「テクニカルとファンダメンタルズの融合」を実装した、完全自律型の日本株シミュレーショントレードBOTです。
東証の全銘柄（約4000銘柄）から資金流入の初動を検知し、ハイブリッドAI（Gemini & Groq）がテーマ性やニュースを加味して自動で精査・仮想売買・資金管理を行います。

## 🔥 特徴・搭載ロジック

### 1. 高精度テクニカルスクリーニング

東証全銘柄の15分足データを高速スキャンし、人間が見逃す「急騰の初動」を以下の条件で機械的に足切りします。

* **流動性・低位株フィルター**: 株価100円未満、または15分平均売買代金が1000万円未満の過疎株・クズ株を排除し、スリッページを防ぎます。
* **資金流入検知**: 過去の平均出来高に対して3倍以上の出来高急増（大口の介入）。
* **ボラティリティ・ブレイク**: ボリンジャーバンドのスクイーズ（収縮）からのエクスパンション（急拡大）。
* **VWAPフィルター**: 現在値がVWAP（出来高加重平均価格）を上回っているか（騙し下げの回避）。
* **上ヒゲ回避**: 高値掴みを防ぐため、ATR（ボラティリティ）を基準に長すぎる上ヒゲを排除。

### 2. ハイブリッドAIスコアリング（Gemini & Groq）

スクリーニングを通過した「資金流入率上位50銘柄」に対し、AIがプロの機関投資家として振る舞い、1〜100点で厳格にスコアリングします。

* **テーマ性と材料の評価**: 取得した最新ニュース（Google News）から、「資金流入の信憑性」「テーマの持続性」を評価し、ニュースのない仕手株のダマシを減点します。
* **APIフォールバックシステム**: メインのAI（Gemini 2.5 Flash）がレートリミット（利用制限）に達した場合、自動的に超高速・無料のAI（Groq Llama 3.3）に通信を切り替えて審査を続行する、無停止のハイブリッドシステムを搭載。

### 3. プロ仕様のリスク管理（ATR連動ストップ）と資金管理

* **動的トレーリングストップ（ATR）**: 固定のパーセンテージではなく、銘柄ごとのボラティリティ（ATR）を基準に利確・損切りラインを自動計算。「値動きの激しい株のノイズで狩られる」ことを防ぎ、損小利大を極限まで追求します。
* **ミニ株（1株単位）のフルインベストメント**: 100株単位の縛りをなくし、指定した予算（例：50万円）の限界まで1株単位で買い付けることで、複利効果を最大化します。
* **地合いフィルター**: 日経平均（^N225）の15分足チャートを確認し、短期下落トレンド時は新規エントリーを一時見送ります。

### 4. 完全自律稼働（営業時間フィルター）

* プログラム自身が現在時刻を取得し、**平日の東証取引時間（9:00〜11:30、12:30〜15:30）のみ稼働**します。夜間や土日に無駄にAPI枠を消費することはありません。

## 🚀 使い方・セットアップ

### 1. 必要なライブラリのインストール

Python環境が構築されているPCで、以下のコマンドを実行し必要なパッケージをインストールします。

```bash
pip install yfinance pandas numpy google-genai feedparser groq python-dotenv

```

### 2. APIキーの設定（.envファイルの作成）

`auto_trade.py` と同じフォルダ内に `.env` という名前のファイルを作成し、以下の形式でAPIキーを記述して保存してください。

```env
GEMINI_API_KEY=あなたのGeminiのAPIキーを入力
GROQ_API_KEY=あなたのGroqのAPIキーを入力

```

> ※GroqのAPIキーは [Groq Cloud](https://console.groq.com/) から無料で取得できます。

### 3. プログラムの実行

コマンドプロンプトやターミナルからスクリプトを実行します。

```bash
python auto_trade.py

```

### 4. 完全自動化（タスクスケジューラの設定）

Windowsの「タスクスケジューラ」やMacの「cron」を利用し、本スクリプトを **1時間ごと** に自動実行するように設定することで、完全放置のシミュレーション環境が完成します。（プログラム内に営業時間フィルターがあるため、24時間設定のままで問題ありません）

💡 **プロの運用Tips**
実行時間は「毎時00分」ではなく、**「毎時16分」「毎時31分」「毎時46分」** などに設定してください。（例：10時16分）
これにより、「10:00〜10:15」の15分足ローソク足が完全に確定した直後の、正確なチャートデータを読み込んで分析することができます。

## ⚙️ パラメーター設定（カスタマイズ）

運用結果を見てロジックを調整したい場合は、`auto_trade.py` 上部の以下の変数を変更してください。

```python
INITIAL_CASH = 1000000          # 初期の仮想資金（円）
MAX_POSITIONS = 3               # 同時に保有できる最大銘柄数
INVEST_PER_TRADE = 500000       # 1銘柄あたりの投資上限（円）

# ATR（ボラティリティ）ベースの動的ストップ設定
ATR_STOP_LOSS_MULTIPLIER = 2.0  # 買値からATRの何倍下がったら損切りするか
ATR_TRAIL_ACTIVATION = 1.5      # 買値からATRの何倍上がったらトレール発動（利確準備）するか
ATR_TRAIL_STOP_MULTIPLIER = 1.5 # 最高値からATRの何倍下がったら利確実行するか

MAX_AI_CANDIDATES = 50          # 1回の実行でAIに審査させる最大銘柄数（1時間ごとなら50で無料枠内に収まります）

```
